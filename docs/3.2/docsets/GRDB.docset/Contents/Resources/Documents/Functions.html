The following functions are available globally.

  • splittingRowAdapters(columnCounts:)

    Returns an array of row adapters that split a row according to the input number of columns.

    For example:

    let sql = "SELECT 1, 2,3,4, 5,6, 7,8"
//               <.><. . .><. .><. .>
let adapters = splittingRowAdapters([1, 3, 2])
let adapter = ScopeAdapter([
    "a": adapters[0],
    "b": adapters[1],
    "c": adapters[2],
    "d": adapters[3]])
let row = try Row.fetchOne(db, sql, adapter: adapter)
row.scopes["a"] // [1]
row.scopes["b"] // [2, 3, 4]
row.scopes["c"] // [5, 6]
row.scopes["d"] // [7, 8]

    Declaration
